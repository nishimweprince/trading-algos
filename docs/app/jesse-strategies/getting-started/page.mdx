# Getting Started

This guide covers setting up the Jesse trading framework and running the AMT strategies.

## Prerequisites

- Python 3.9+
- PostgreSQL (for live trading)
- Docker (optional, for containerized deployment)

## Installation

### Option 1: Local Installation

```bash
# Install Jesse
pip install jesse

# Navigate to project
cd jesse-strategies

# Install additional dependencies
pip install -r requirements.txt
```

### Option 2: Docker Installation

```bash
# Navigate to project
cd jesse-strategies

# Start with Docker Compose
docker-compose up -d

# Access dashboard
open http://localhost:9000
```

## Configuration

### Exchange Setup

Edit `config.py`:

```python
config = {
    'exchanges': {
        'Binance Spot': {
            'fee': 0.001,
            'type': 'spot',
            'balance': 10000,  # For backtesting
        },
    },
    'app': {
        'debug_mode': False,
        'trading_mode': 'backtest',  # or 'paper' or 'live'
    },
}
```

### Strategy Routes

Edit `routes.py`:

```python
from jesse.strategies.AMTTrendContinuation import AMTTrendContinuation

routes = [
    ('Binance Spot', 'BTC-USDT', '1h', AMTTrendContinuation),
]
```

## Importing Historical Data

Before backtesting, import candle data:

```bash
# Import BTC-USDT data from Binance
jesse import-candles "Binance Spot" BTC-USDT 2023-01-01

# Import multiple pairs
jesse import-candles "Binance Spot" ETH-USDT 2023-01-01
jesse import-candles "Binance Spot" SOL-USDT 2023-01-01
```

## Running a Backtest

```bash
# Basic backtest
jesse backtest 2023-01-01 2024-01-01

# With debug mode
jesse backtest 2023-01-01 2024-01-01 --debug

# Export results
jesse backtest 2023-01-01 2024-01-01 --csv
```

## Live Trading Setup

<Callout type="warn">
**Warning**: Live trading involves real financial risk. Always test thoroughly in paper trading mode first.
</Callout>

1. Configure API keys in environment:

```bash
export JESSE_BINANCE_API_KEY="your_api_key"
export JESSE_BINANCE_API_SECRET="your_api_secret"
```

2. Update config for live trading:

```python
config = {
    'app': {
        'trading_mode': 'live',
    },
}
```

3. Start live trading:

```bash
jesse live
```
